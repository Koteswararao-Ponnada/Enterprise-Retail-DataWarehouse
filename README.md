# Enterprise Retail Data Warehouse

## 📌 Project Overview

This project demonstrates how an Enterprise Retail Data Warehouse is designed and implemented using modern Data Engineering technologies.

The project starts from an OLTP database and gradually builds a complete Data Warehouse with ETL pipelines, Python automation, PySpark transformations, Databricks notebooks, and reporting.
